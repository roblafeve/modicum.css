modicum.css

a tweetable CSS framework

Modicum is an amazingly terse & reliable CSS layout system.

How it works.

Modicum.css uses 'display: table' and 'display: table-cell' to mimic what tables have done from the start— automatic column widths (...settle down, we're only letting the browser apply similar layout rules. There is no effect on an element's semantic value.) This means that you simply create a new column and move on. No percentage discrepancies between browsers. No clearing floats. Matching height columns defined by content (yay!)

Application.

With Modicum, rows and columns can be infinitely nested within each other (it's not prctical— but go ahead and give it a try.) Because of this, Modicum is useful for layouts and modules alike. An entire web app could be defined using 97 characters, or 98B of css. I like the sound of that!

Fully responsive.

Responsive layouts? Yep, just switch any/all column elements to 'display: block' at the desired breakpoint.

Code example:

[row] {
 display: table;
 table-layout: fixed;
 border-collapse: collapse;
 width: 100%;
}

[column] {
 display: table-cell;
 vertical-align: top;
}
